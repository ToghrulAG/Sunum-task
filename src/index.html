Sass/Scss Functions

SCSS-dəki funksiyalar parametrləri qəbul edə, hesablamalar apara və nəticə qaytara bilən kod parçalarıdır. Onlar kodun təkrar istifadəsinə imkan verir və daha davamlı üslubu təşviq edir. Funksiyalar kodun təkrarlanmasının qarşısını alaraq onu daha oxunaqlı və davamlı edir. Çeviklik və Parametrlər: Funksiyalar parametrləri qəbul edə bilər, onları müxtəlif kontekstlərə çevik və uyğunlaşdırır. Modulluq: Funksiyalardan istifadə modul və strukturlaşdırılmış kodu təşviq edir.

Code

İlk baxışdan funksiyalar mixinlərə çox bənzəyir. Parametr qəbul etmə kimi oxşar funksiyaları olsa belə, funksiyalar yanlız bizə return olaraq yanlız və yanlız dəyər qaytarır. Hansıki o dəyəri çevirmənin istənilən yerində istifadə edə bilərik.

@function calculateWidth($width, $margin) {
    @return $width + $margin;
}
      
.example {
    width: calculateWidth(100px, 20px);
}

Gəlin qradiyent yaratmaq üçün funksiyalardan istifadə nümunəsinə baxaq:

Bu nümunə, generateGradient funksiyasının kodunuzu necə təmizlədiyini və gələcəkdə dəyişiklik etməyi asanlaşdırdığını nümayiş etdirir. 

    @function generateGradient($color1, $color2) {
        @return linear-gradient(to right, $color1, $color2);
      }
      
      .header {
        background: generateGradient(#3498db, #2ecc71);
      }

Nəticə olaraq, SCSS-dəki funksiyalar davamlı, çevik və səmərəli üslub kodu yaratmaq üçün güclü bir alət təmin edir. Onların istifadəsi tərtibatçılara inkişaf vaxtını azaltmağa, kodun oxunuşunu artırmağa və layihənin modulluğunu təmin etməyə imkan verir
